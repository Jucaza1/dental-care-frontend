DentCare

About this project

The aim of this project is to elaborate a web page for a dental clinic.

This application consists of three main components:

  • A Go backend for file serving.
  • The HTML and CSS static file structure.
  • The TypeScript compiled and bundled with Webpack.

The storage is made on local cookies that stay on the browser.

Future features like data exchange with the backend and user login are on the way to be added.
